# 마녀공장 리디자인 | ma:nyo

퍼블리셔 취업 준비를 하면서 마녀공장(MANYO FACTORY) 공식 쇼핑몰 UI를 참고해서 처음부터 직접 퍼블리싱해본 연습용 포트폴리오입니다.

> 이 프로젝트는 (주)마녀공장과 관련이 없는 개인 학습/포트폴리오용 사이트이며, 상업적 목적이 없습니다. 실제 브랜드 이미지를 학습용으로 참고했습니다.

## 데모

- 배포 링크: https://dohyeonl.github.io/manyo-redesign/

## 사용 기술

- HTML5 / CSS3 (Flexbox, Grid, 반응형 미디어쿼리)
- Vanilla JavaScript
- Swiper.js (상품 캐러셀)
- Google Fonts

## 주요 기능

- 반응형 레이아웃 (PC / 태블릿 / 모바일 대응, 768px 이하에서는 햄버거 메뉴로 전환)
- Swiper로 만든 Best Product 캐러셀 + 진행바
- 추천 상품 필터 버튼 (MD추천 / 프로모션 특가 / 꿀조합 SET)
- IntersectionObserver로 스크롤하면 콘텐츠가 페이드인되는 효과
- 키보드로 Tab 눌러도 드롭다운 메뉴가 열리도록 처리
- 검색 아이콘 클릭 시 검색창 토글

## 실제 사이트와 비교해서 다르게 만든 점

그냥 화면을 똑같이 따라 그린 게 아니라, 실제 마녀공장 사이트를 써보면서 불편했던 부분을 찾아서 다르게 설계했습니다.

접속하자마자 화면 절반을 가리는 회원가입 팝업이 뜨고, 히어로 이미지 위에는 "1초 회원가입" 배지가 어색하게 겹쳐 있어서 제품을 보기도 전에 방해 요소부터 만나게 됩니다. 리디자인에는 이런 팝업이나 겹치는 배지 없이 브랜드 배너와 제품이 바로 눈에 들어오도록 했습니다.

히어로도 원본은 브랜드 소개와 상품 광고 슬라이드가 섞여서 자동으로 돌아가는 캐러셀이라 접속할 때마다 처음 보이는 화면이 달라집니다. 배너를 하나씩 눌러보니 실제로는 대부분 상품 구매 페이지(`goods_view.php`)로 연결돼서, 이벤트처럼 보이지만 그냥 상품 광고였습니다. 그래서 고정된 브랜드 배너 하나만 두어서 첫인상이 항상 같게 유지되도록 만들었습니다.

진짜 이벤트/프로모션은 EVENT 카테고리를 따로 눌러야만 나오는 별도 게시판에만 있고 메인 화면에는 전혀 안 보입니다. 상품을 보여주는 방식도 "지금 가장 사랑받는 제품" 섹션 하나뿐이었습니다. 그래서 할인 중인 제품과 세트 상품을 필터로 골라볼 수 있는 Recommended Products 섹션을 메인에 새로 만들어 넣었습니다.

카테고리 메뉴 접근성도 확인해봤는데, 마우스로 호버하거나 클릭해야만 하위 카테고리(스킨케어, 클렌징 등)가 펼쳐지고 키보드 Tab만으로는 포커스 자체가 그 항목까지 가지 않았습니다. CSS에 아래처럼 `focus-within` 규칙을 추가해서, 마우스 없이 Tab 키만으로도 하위 카테고리를 확인할 수 있도록 만들었습니다.

```css
/* 호버하면 드롭다운 보임, focus-within은 탭키로도 열리라고 추가한거 */
.nav-item:hover .dropdown,
.nav-item:focus-within .dropdown {
  opacity: 1;
  pointer-events: auto;
}
```
